# Signal-Simulator-Demo-Mode

WORKS IN KERNELSU!

datatype that can be placed: 1x, e, g, h, h+, 3g,3g+, 4g, 4g+, 5g, 5g+, 5ge
(5g, 5g+ and 5ge only works on some modern ROMs)

Recommended Android versions: those compatible with GravityBox, e.g., Android 10, Android 11, etc.

English:


Magisk module that simulates signal and network by taking advantage of the Android demo mode


NOTICE: THIS MODULE DOES NOT PROVIDE A REAL NETWORK OR SIGNAL; IT TAKES ADVANTAGE OF ANDROID'S DEMO MODE.

The module can now be managed with commands in Termux; run minenet -h to see the commands.
